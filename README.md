* Present

    * ExpressJS, NodeJS: Backend (whole website collaborates with SQL, etc ){BusinessLogic platform}

### Set up React project 
```bash
node -v 
npm create vite@latest 
### To run web server 
npm run dev # make sure that you cd to project directory 

### install bootstrap 
npm i bootstrap 
### import bootstrap in main.tsx
import 'bootstrap/dist/css/bootstrap.css'
```
### Explore what is react-app
